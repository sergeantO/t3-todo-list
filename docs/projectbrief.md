# Project Brief: Todo List Application

*Последнее обновление: 2023-01-10*

## Обзор проекта
Проект представляет собой приложение для управления задачами (Todo List), построенное на стеке T3 (Next.js, Prisma, tRPC, Tailwind CSS и NextAuth.js). Основные характеристики:
- Полноценное full-stack приложение с end-to-end type safety
- Аутентификация через NextAuth.js
- Управление состоянием через React Query
- ORM Prisma для работы с базой данных

## Основные функциональные требования 
- CRUD операции для задач
- Система аутентификации пользователей
- Валидация данных через Zod
- Управление состоянием клиента через React Query
- Серверная логика через tRPC

## Структура проекта 
```
t3-todo-list/
├── prisma/           # Схемы БД и миграции
├── src/              # Исходный код
│   ├── pages/        # Next.js страницы
│   ├── server/       # tRPC роутеры
│   └── styles/       # Глобальные стили
├── public/           # Статические ресурсы
├── tasks/            # Задачи
├── docs/             # Документация
└── generated/        # Автогенерируемые файлы
```
